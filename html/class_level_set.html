Detailed Description

The class of the Level set field.

This class contains most of the methods used in the program, such as calculating the reference contact angle, calculating the contact angle from the simulation data, tracking the contact point and evolving the field with time.

array< int, 3 > LevelSet::getContactPointIndices ( array< double, 3 >  point)

Return the indices of the contact point in 2D or the coordinates most closely matching the given parameter.

This function works differently for 2D and 3D. If the simulation is 2D, it ignores the input parameter and returns indices of the contact point by checking where the sign of the LevelSet field changes. In 3D, it returns the coordinates of the point on the grid which is closest to the parameter.

Parameters
pointA point in the space of the Level set field.
Returns
2D: Indices of the contact point. 3D: Indices of the point closest to point.
